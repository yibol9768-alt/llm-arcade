# SUPER MARIO · 5.6 SOL

一个零依赖、离线可玩的 HTML5 Canvas 横版平台小游戏。项目中的画面、动画、关卡和声音均由代码生成，不依赖外部图片、字体或音频文件。

## 启动

在当前目录运行：

```bash
python3 -m http.server 8560
```

然后浏览器访问 `http://127.0.0.1:8560`。

也可以直接双击 `index.html` 游玩。

## 操作

- `← →` 或 `A D`：移动
- `Space`、`W` 或 `Z`：跳跃
- `X` 或 `J`：获得火焰能力后发射火球
- `P` 或 `Esc`：暂停 / 继续
- `M`：声音 / 静音
- `R`：重新开始

## 游戏内容

- 完整横版关卡、断崖、隐藏金币路线、问号砖、可破坏砖块、管道、阶梯和终点城堡
- 蘑菇与火焰花成长系统
- Goomba 风格敌人、Koopa 风格敌人及龟壳互动
- 跑动、跳跃、踩踏、受伤、无敌闪烁、粒子、镜头震动、烟花等反馈
- Web Audio 实时合成背景音乐与音效，支持按钮和 `M` 键一键静音
- 键盘与触屏双操作，自动适配窗口比例
- 本地最高分记录
