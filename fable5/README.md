# Super Mario JS — Fable-5 Build

单文件 HTML5 马里奥式平台跳跃游戏,零依赖、零外部资源。双击 `index.html` 即玩(或任意静态服务器)。

## 特性

- **NES 风格 1-1 关卡**:开局问号砖阵、四根水管、三处深坑、地下砖列、双金字塔、终点 8 级阶梯 + 旗杆 + 城堡,布局按原版 1-1 节奏复刻
- **像素美术全部代码手绘**(ASCII 像素图 → canvas),非拷贝任天堂素材;256×240 内部分辨率,3 倍整数缩放,零模糊
- **完整机制**:变加速跑动/打滑、可变高度跳跃、踩敌、顶砖(小马里奥顶起 / 大马里奥击碎)、问号块出金币与蘑菇、吃蘑菇变大、受击缩小 + 无敌闪烁、乌龟踩成龟壳可踢、龟壳连杀、滑旗结算、时间换分、命数与 GAME OVER、TOP 分数本地存档
- **声音全部 Web Audio 实时合成**:NES 式占空比方波 + 三角波贝斯 + 噪声鼓组;BGM 为本项目**原创曲**(非任天堂旋律,YouTube 不会触发 Content ID);跳跃/金币/踩踏/顶砖/变大/死亡/滑旗全套音效
- **静音方式**:`M` 键或画面下方 SOUND 按钮,状态持久化

## 操作

| 键 | 功能 |
|---|---|
| ← → / A D | 移动 |
| Z / Space / ↑ | 跳(按住跳更高) |
| X / Shift | 加速跑 |
| Enter | 开始 |
| P | 暂停 |
| M | 静音/开声 |

## 调试参数(录视频好用)

- `?state=play` 跳过标题直接开局(内置演示 AI 自动向右跑跳)
- `&ff=N` 快进 N 帧
- `&sx=<像素>` 传送到关卡任意位置(旗杆约 3100)
- `&god=1` 无敌
- `&big=1` 直接大马里奥
- `?selftest=1` 运行 12 项机制自检(金币、蘑菇、变大、碎砖、踩踏、死亡、顶砖、龟壳三连)

## 技术

单文件约 1500 行:5×7 位图字体、周期波形音频调度器(前瞻 140ms)、瓦片 AABB 分轴碰撞、行程编码关卡 + 装饰循环、定步长 60Hz 主循环。无框架、无构建、无网络请求。
