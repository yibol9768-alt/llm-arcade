# 5.6Luna: Mushroom Run

A self-contained pixel-art platform game made for the `test/5.6luna` folder.

## Controls

- Move: `Arrow keys` or `A / D`
- Jump: `Space`, `W`, or `Arrow Up`
- Pause: `P` or `Escape`
- Sound: `M`, or the Sound button in the top-right corner

The game uses original canvas-drawn visuals and small Web Audio effects, so it works offline with no asset downloads. Open `index.html` in a modern browser and press **Start adventure**.
